Para quem é este curso?

Qualquer pessoa que procure avançar no entendimento de CSS (conheça o básico de HTML e CSS) para desenvolver layouts responsivos. Se você já tem experiência com HTML e CSS vai simplesmente completar o curso mais rápido.

  • Desenvolvedores que utilizam frameworks como Bootstrap e queiram saber como tudo funciona por trás das classes
  • Desenvolvedores preocupados em acompanhar as práticas modernas de layout que estão sendo adotadas
  • Qualquer pessoa que queira melhorar seu nível de CSS e aprender como criar suas próprias ferramentas, usando conceitos básicos e avançados na prática
  • Designers e desenvolvedores interessados em saber como criar sites responsivos
